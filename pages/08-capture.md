# 화면 캡처

**화면 캡처** 탭에서 AI가 «보고 있는 화면»을 설정합니다. 멀티모달 LLM이 활성화되면 캡처 이미지가 응답 생성에 포함됩니다.

## 캡처 모드

| 모드 | 설명 |
|------|------|
| 카메라/화면 | 웹캠 또는 특정 창·모니터 캡처 |
| 직접 입력 | 오버레이 창에 텍스트·이미지를 직접 입력 |

## 주요 설정

- **이미지 인식 사용** — 켜면 응답 시점에 화면 캡처를 LLM에 전달
- **캡처 대상** — 창 제목 또는 모니터 선택
- **캡처 영역 표시** — 선택 영역을 화면에 테두리로 표시
- **크로마키** — 실행 제어 패널에서 Unity 배경과 연동

## 직접 입력 모드

1. 캡처 모드를 **직접 입력**으로 선택합니다.
2. **캡처 영역 표시**를 누르면 오버레이 창이 나타납니다.
3. 오버레이에 텍스트를 입력하면 AI가 그 내용을 «본 것»처럼 반응할 수 있습니다.

[[TIP("TIP")]]
카메라 모드에서 직접 입력으로 바꿀 때 오버레이가 자동으로 표시됩니다.
[[/TIP]]
## Chroma (크로마키)

Live2D를 방송 소스로 쓸 때 배경을 단색(그린)으로 맞추는 설정입니다. Unity **크로마키** 버튼과 함께 사용합니다.

## 실행 중 반영

캡처 모드·이미지 인식·캡처 대상 변경은 **프로그램 실행 중에도 즉시 반영**됩니다.
