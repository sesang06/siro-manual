## Idle 대화 (자동 발화)

| UI 라벨 | 설명 | 즉시 반영 |
|---------|------|-----------|
| 자동 발화 활성화 | 침묵 시 캐릭터가 먼저 말함 | O |
| 기본 침묵 임계값 (초) | 이 시간 조용하면 발화 후보 | O |
| 랜덤 편차 (%) | 임계값에 난수 적용 | O |

AI가 말하는 중·마이크 VAD·대기 댓글/후원이 있으면 Idle은 발동하지 않습니다.

장기 메모리 사용 시 Qdrant(`localhost:6333`)가 필요합니다. [[04. 문제 해결|문제 해결]] 참고.
