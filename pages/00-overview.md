# 시로 AI 버튜버 사용 설명서

시로 AI 버튜버는 **Python 백엔드(대화·STT·TTS)** 와 **Unity Live2D 클라이언트**가 함께 동작하는 AI 버튜버 방송 도구입니다.

[[TIP("시작하기")]]
스크린샷 포함 상세 절차는 **01. 시로 AI 소개** → **02. 실행 방법 안내** 순서로 읽어 주세요.
[[/TIP]]

## 구성 요소

| 구성 | 역할 |
|------|------|
| **AI 버튜버 실행 프로그램** | LLM 대화, 음성 인식(STT), 음성 합성(TTS), 채팅·후원 연동 |
| **Unity 클라이언트 (siro-launcher)** | Live2D 캐릭터 표시, 자막, 오디오 재생 |
| **GPT-SoVITS** (선택) | 로컬 TTS 엔진 — `tts 모듈` 사용 시 자동 기동 |

## 다운로드

Unity 클라이언트는 별도 준비가 필요합니다. **07. Unity 클라이언트** 페이지를 참고하세요.

## 빠른 시작

1. **01. 시로 AI 소개** — 다운로드·요구조건
2. **02. 실행 방법 안내** — API 키·음성·화면 인식
3. Unity 클라이언트 실행 → Python **프로그램 시작**
4. 마이크로 말하면 AI가 응답합니다
