# 02-9. 실행 제어

우측 **실행 제어** 패널입니다. 프로그램 **실행 중에도 즉시 반영**됩니다.

![실행 제어 패널](../assets/ui/panel-runtime-control.png)

| UI 라벨 | 설명 |
|---------|------|
| 채팅 열기 | 텍스트로 AI와 대화 (테스트) |
| 사용 설명서 | WikiDocs 메뉴얼 열기 |
| 크로마키 ON/OFF | Unity 배경 그린 스크린 모드 |
| AI 발화 중단 | AI TTS 일시 중지 (실행 중에만 활성) |
| 마이크 뮤트 | STT 입력 끄기 |
| 방송인이 말하는 동안에 AI가 말하지 않음 | STT 감지 시 AI TTS 중단 |
| AI 발화 중 마이크 음소거 | AI 발화 중 STT 입력 중지 |
| 프로그램 시작 | STT·TTS·채팅 모니터 등 백엔드 기동 |

## 사용 순서

1. [[01. 시작하기|Unity 클라이언트]] 실행
2. AI·캐릭터·오디오 설정 확인
3. **프로그램 시작** 클릭
4. 마이크로 말하거나 **채팅 열기**로 테스트

정상 연결 시 로그: `Connected to server at ws://localhost:8765`

문제가 있으면 [[04. 문제 해결|문제 해결]]을 참고하세요.
