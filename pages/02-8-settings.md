# 02-8. 설정 관리

![설정 관리 패널](../assets/ui/panel-settings.png)

| UI 라벨 | 설명 | 즉시 반영 |
|---------|------|-----------|
| 설정 내보내기 | `config.yaml`·캐릭터 yaml zip 백업 | — |

업데이트 전 백업·다른 PC로 옮길 때 사용합니다.

[[WARNING("WARNING")]]
가져오기는 기존 설정을 덮어씁니다. 필요하면 먼저 내보내기로 백업하세요.
[[/WARNING]]

대부분의 UI 항목은 다른 탭에서 변경 시 **자동 저장**됩니다.  
고급 사용자는 프로그램 **종료 후** `setting_doc/config.yaml`을 직접 편집할 수 있습니다.

버전 확인: 앱 하단 또는 `version.yaml` — [[04-1. 변경 이력|변경 이력]]과 대조.
