# 설정 관리

**설정 관리** 탭에서는 UI 설정 파일을 백업·복원합니다.

## 설정 내보내기

1. **설정 내보내기** 클릭
2. 저장할 zip 경로 선택
3. `config.yaml`, 캐릭터 yaml 등이 포함된 아카이브 생성

업데이트 전 백업·다른 PC로 옮길 때 사용합니다.

## 설정 가져오기

1. **설정 가져오기** 클릭
2. 이전에 내보낸 zip 선택
3. `setting_doc` 등이 복원됩니다

[[WARNING("WARNING")]]
가져오기는 기존 설정을 덮어씁니다. 필요하면 먼저 내보내기로 백업하세요.
[[/WARNING]]
## 자동 저장

대부분의 UI 항목은 변경 시 `config.yaml` / 캐릭터 yaml에 **자동 저장**됩니다. 별도 «저장» 버튼이 없는 탭이 많습니다.

## 수동 편집

고급 사용자는 `setting_doc/config.yaml`, `setting_doc/<캐릭터>/config.yaml`을 직접 편집할 수 있습니다. 프로그램이 **실행 중이 아닐 때** 편집하는 것이 안전합니다.

## 버전 정보

앱 하단 또는 `version.yaml`에서 현재 버전을 확인할 수 있습니다. [[16. 변경 이력|변경 이력]]과 대조하세요.
