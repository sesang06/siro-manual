## 위플랩 채팅 설정

![screenshot](../assets/gdoc/img_052.png)

[위플랩 채팅](https://weflab.com/chat) → **채팅창** → **URL** 복사.

![screenshot](../assets/gdoc/img_053.png)

- **위플랩 채팅 댓글 감지 및 반응** 체크
- URL 입력 후 [[02-2. 캐릭터|캐릭터]]에서 시청자 대화 시나리오 선택

AI가 말하는 중에는 댓글이 큐에 쌓였다가 TTS 종료 후 처리됩니다. 스트리머 발화(VAD) 중에는 건너뜁니다.

## 위플랩 후원

![screenshot](../assets/gdoc/img_054.png)

후원은 누적 댓글보다 **우선** 처리됩니다.

## 상세 설정 다이얼로그

| 항목 | 설명 |
|------|------|
| 샘플링률 | 댓글 일부만 무작위 반응 |
| 의미 없는 채팅 필터 | 짧은·무의미 메시지 제외 |
| 차단 키워드·닉네임 | 블록 리스트 |
| 닉네임 필터 | 닉네임 치환 |
| 큐 최대 크기 | 한 번에 처리할 댓글 수 |
| 댓글 오버레이 출력 | Live2D 말풍선 |
| 닉네임 언급 금지 | 닉네임 필터링 |

![screenshot](../assets/gdoc/img_056.png)

`show_browser` — Playwright로 채팅/후원 페이지 표시 (디버그).  
`react_to_comments_immediately` — 댓글 즉시 큐 투입 (`config.yaml`).

## 오버레이

| 요소 | 내용 |
|------|------|
| **자막** | AI **말하는 대사** (TTS 동기) |
| **오버레이** | **시청자 채팅·후원** 말풍선 |

AI가 댓글/후원에 **응답하기 시작할 때** Unity로 overlay 이벤트가 전송됩니다.  
OBS 창 캡처 시 자막·오버레이가 함께 녹화됩니다.

문제가 있으면 [[04. 문제 해결|문제 해결]]의 댓글/후원 항목을 참고하세요.
